# opencv-image-processing-exercises

This repository contains weekly exercises for learning image processing using Python, OpenCV, and NumPy.

## Exercises 1

1. Color channels and grayscale
2. Mini image editor (brightness, contrast, threshold)
3. Image resolution scaling
4. Synthetic image generation

## Exercises 2

1. Adaptive brightness adjuster
2. Noise detector filter
3. Smart blur selector

## Tech Stack

- Python
- OpenCV
- NumPy
